CPSC 314 Final Project Presentation

Exerlog

Lucas Abeln

Exerlog is a website that allows you to keep track of workouts! It is unique to each user and allows an array of options
for sorting and organizing your workouts.

Functional Requirements:

  1. Must be able to use a dropdown to sort between workout categories

  2. Must be able to log in to a user profile or create a new username

  3. Must be able to create a new work out log

  4. Users must be able to toggle sorting logs by date

  5. Users must be able to use calendar to access certain months logs

  6. Must log data from logs in a JSON

  7. Must be able to delete a workout log

  8. Must store unique data under each user login for retrieval upon login

  9. Must compute average time spent exercising for display

  10. Must set goals for the users next week
